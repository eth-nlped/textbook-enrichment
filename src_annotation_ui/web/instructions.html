Textbook Enrichment Evaluation
(Annotation Guidelines))
These guidelines are intended for an evaluation study at ETH Zurich. You will be evaluating the performance of an AI-based method that uses web images to assign images to textbook sections. You will be presented with text from a section of a textbook, which is divided into multiple subsections. Each image will be accompanied by questions that ask about the relationship between the text and the image, as well as the relationship between the image and other images in the subsection. At the end of the evaluation, you will be shown all the images again and asked some section-level questions. The goal of this study is to create more engaging and effective learning resources for students. In specific, with each image assigned to a subsection, you'll be posed with following questions:
  • Is this image relevant to educational concepts described in the subsection shown here?
  • Is this image redundant as compared to images shown previously in this subsection?
  • Is this image didactically useful for explaining the concept(s) described in subsection text?
  • What is the type of this image?
  • Is this image relevant to the educational concepts described in the section shown here?
  • Is this image redundant as compared to the images shown previously at the end of this section? select one: (a) yes; (b) no; (c) undecided.
  • Is this image didactically useful for explaining the concept(s) described in section text?
Definition of Options (will also be shown during the annotation) Options concerning relevancy questions:
  • relevant: some concept described in the subsection text is picturised in the image;
  • somewhat-relevant: image is loosely on the same subject/topic domain as the subsection text, but is not exactly about the same concepts;
  • irrelevant: not at all related to the subsection text.
  • undecided: unable to make a decision, maybe because of my low understanding of the text or image.
Options concerning redundancy questions:
  • yes: this image illustrates exactly the same concept as one of the previously shown images in the subsection (or section, if end-of-section evaluation);
  • no: this image illustrates atleast some unique concepts as compared to the previously shown images in the subsection (or section, if end-of-section evaluation);
  • undecided: unable to make a decision, maybe because I do not understand the concepts illustrated by this image.
Options concerning didactic usefulness questions:
  • yes: image will be appropriate for teaching the section to a student - I would like it here (with an appropriate caption) if I was learning from this textbook;
  • no: image is not appropriate for teaching the section text;
  • undecided: unable to make the decision, maybe because of my less understanding of the text or image.
Please answer the questions as they appear in the section layout, since order is important for redundancy related questions. Layout of a Example Section

  • learning objectives of this section
  • text from subsection 1
  • image 1 and subsection-level questions
  • image 2 and subsection-level questions
  • text from subsection 2
  • image 3 and subsection-level questions
  • text from subsection 3
  • text from subsection 4
  • image 4 and subsection-level questions
  • image 5 and subsection-level questions
